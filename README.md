## 🎯 Bot Commands

### ⚙️ Basic Commands
- `!help` — Show all commands  
- `!menu` — Show bot menu  
- `!ping` — Check bot speed  
- `!owner` — Bot owner info  
- `!info` — Bot details  

---

### 📥 Media Download
- `!yt [url]` — Download YouTube video  
- `!ig [url]` — Instagram download  
- `!tt [url]` — TikTok download  
- `!fb [url]` — Facebook download  
- `!spotify [url]` — Spotify audio  

---

### 🤖 AI Features
- `!ai [question]` — Chat with AI  
- `!gpt [prompt]` — GPT assistant  
- `!img [prompt]` — AI image generator  
- `!translate [text] [lang]` — Translate text  
- `!summarize [text]` — Summarize content  

---

### 🧰 Utilities
- `!sticker` — Convert image to sticker  
- `!qr [text]` — Generate QR code  
- `!weather [city]` — Get weather info  
- `!calc [expression]` — Calculator  
- `!currency [amount] [from] [to]` — Currency converter
